# Contributing

Thanks for improving Profile Card Forge.

## Good first changes

- Add a new theme preset
- Improve keyboard accessibility
- Add a new Markdown layout
- Fix mobile spacing issues
- Improve copy/download feedback

## Pull request checklist

- Keep the app static and dependency-free unless there is a clear reason.
- Test in a desktop and mobile-sized browser window.
- Keep generated Markdown readable.
- Update the README when behavior changes.

## Issue guidelines

Please include:

- What you expected
- What happened
- Browser and operating system
- Screenshots when useful
